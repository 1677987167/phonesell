                手机售卖管理系统 — 生成 APK 方法
══════════════════════════════════════════════════════════════

本系统已改造为纯前端应用（IndexedDB 存储数据），
无需后端服务器即可运行，完全离线可用。

以下是将本系统安装到 Android 手机的几种方法：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方法一（推荐）：Chrome 添加到主屏幕（无需 APK）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 启动本系统的服务器（运行 启动服务.bat）
2. 手机和电脑连接同一 WiFi
3. 手机 Chrome 浏览器访问 http://电脑IP:3000
4. 点 Chrome 菜单 → 添加到主屏幕
5. 桌面上会出现图标，像原生 App 一样使用

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方法二：PWABuilder 在线生成 APK
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 将 public 文件夹部署到任何 HTTPS 服务器（可用 GitHub Pages）
2. 打开 https://pwabuilder.com
3. 输入你的网址
4. 点击 "Package for Android"
5. 下载生成的 APK 文件
6. 安装到手机

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方法三：PWA2APK.com 在线生成
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 将 public 文件夹上传到任一静态托管服务
   （推荐：Vercel、Netlify、GitHub Pages，均有免费方案）
2. 打开 https://pwa2apk.com
3. 输入你的 PWA 网址
4. 生成并下载 APK

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方法四：使用 AppMaker.xyz
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 将 public 文件夹部署到 HTTPS 服务器
2. 打开 https://appmaker.xyz/pwa-to-apk
3. 输入你的 PWA 网址
4. 生成 APK 文件

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
注意事项
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
• 所有数据存储在手机的 IndexedDB 中，清除浏览器数据会丢失
  （如需备份，可在服务器启动后使用导入导出功能）
• 默认账户: 123456 / 123456
• 首次启动会自动导入 43 款手机数据
